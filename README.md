# Simple Linux Sniffer (C++/Qt)

Примитивный сниффер для анализа трафика в Linux. Работает через Raw Sockets (AF_PACKET).

### Что умеет:
* Перехват Ethernet-кадров (L2), IP-пакетов (L3), TCP/UDP сегментов (L4).
* Чтение полезной нагрузки (Payload) в текстовом или HEX формате.
* Вывод MAC и IP адресов, портов и типов протоколов.
* Фильтрация трафика (ARP, IPv4, IPv6, TCP, UDP).
* Счетчик общего и отфильтрованного количества пакетов.

### Запуск:
Программе требуются права **root** для открытия сокетов
